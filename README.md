# Vila Activewear

JavaFX desktop shopping app for CST338. Activewear store with a clean look, kind of like Alo or Lululemon.

## Features

- Login and account creation
- Browse products on the home page
- View product details and pick a size
- Shopping cart

## Tech

Java 17, JavaFX 21, Gradle, SQLite

## How to run

Open in IntelliJ IDEA and wait for Gradle to sync. Set the SDK to Java 17+, then run `VilaApp.java`.

From terminal:
```
./gradlew run
```

## Database

Tables: users, products, orders, order_items
